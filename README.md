# RemoteLoader

Qml热加载
## 简介

使用Go语言搭建一个简单的http文件服务器，提供Qml文件。

手机端App通过网络加载Qml文件。

## 目录结构

- App 
   手机端app

- GoServer 
   go语言服务器

- QmlFiles 
   Qml文件
